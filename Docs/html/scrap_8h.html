enum ScrapClipType

The supported scrap clipboard types.

This is only relevant in a X11 environment, which supports mouse selections as well. For Win32 and MacOS environments the default clipboard is used, no matter what value is passed.

int pygame_scrap_put ( char *  type,
int  srclen,
char *  src 
)

Places content of a specific type into the clipboard.

Note
For X11 the following notes are important: The following types are reserved for internal usage and thus will throw an error on setting them: "TIMESTAMP", "TARGETS", "SDL_SELECTION". Setting PYGAME_SCRAP_TEXT ("text/plain") will also automatically set the X11 types "STRING" (XA_STRING), "TEXT" and "UTF8_STRING".

For Win32 the following notes are important: Setting PYGAME_SCRAP_TEXT ("text/plain") will also automatically set the Win32 type "TEXT" (CF_TEXT).

For QNX the following notes are important: Setting PYGAME_SCRAP_TEXT ("text/plain") will also automatically set the QNX type "TEXT" (Ph_CL_TEXT).

Parameters
typeThe type of the content.
srclenThe length of the content.
srcThe NULL terminated content.
Returns
1, if the content could be successfully pasted into the clipboard, 0 otherwise.
