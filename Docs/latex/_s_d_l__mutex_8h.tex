Typical use of condition variables\+:

Thread \mbox{\hyperlink{struct_a}{A}}\+: S\+D\+L\+\_\+\+Lock\+Mutex(lock); while ( ! condition ) \{ S\+D\+L\+\_\+\+Cond\+Wait(cond, lock); \} S\+D\+L\+\_\+\+Unlock\+Mutex(lock);
