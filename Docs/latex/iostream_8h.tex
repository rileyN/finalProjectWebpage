\textbackslash{}rst This is a helper function to add a C++ redirect context manager to Python instead of using a C++ guard. To use it, add the following to your binding code\+:

.. code-\/block\+:: cpp \begin{DoxyVerb}#include <pybind11/iostream.h>

...

py::add_ostream_redirect(m, "ostream_redirect");
\end{DoxyVerb}


You now have a Python context manager that redirects your output\+:

.. code-\/block\+:: python \begin{DoxyVerb}with m.ostream_redirect():
    m.print_to_cout_function()
\end{DoxyVerb}


This manager can optionally be told which streams to operate on\+:

.. code-\/block\+:: python \begin{DoxyVerb}with m.ostream_redirect(stdout=true, stderr=true):
    m.noisy_function_with_error_printing()
\end{DoxyVerb}
